## Projeto Tabuada

Projeto feito ao longo do curso Fullstack Master - [Devpleno](https://devpleno.com/).

### Proposta

Apresentar os conceitos iniciais de [NodeJs](https://nodejs.org/en/) como:

- HTTP Servers
- Rotas, Middlewares e Query strings/params
- Controllers
- Template Engine
